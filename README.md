## General info
A simple application for generating placeholder lorem ipsum text.


## Technologies
* Vanilla Javascript
* HTML5
* SASS


## Features
* User can generate lorem ipsum text as per word count or paragraph count

   *like "20 words" or "3 paragraphs"*
